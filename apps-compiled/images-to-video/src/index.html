Images-to-Video with WebAssembly (Proof-of-Concept)

This was a really quick-and-dirty experimental/prototype project, not necessarily intended for general-use. It's meant to demonstrate the feasibility of using an existing ffmpeg WebAssembly library on npm for converting a series of image files into a single video file.

Upload multiple ".jpg" files below. They will automatically be converted into a video, which should display shortly below. Depending on the browser you use, you should be able to download the file using the video player.

This app *ONLY* accepts ".jpg" files. Everything else is ignored.

NOTE: There is a bug in either ffmpeg or the ffmpeg library which prevents us from receiving accurate progress data. If it says it's done but no video has appeared and no error has occurred, you may have to wait a little bit longer for the video to show up.
